# CoffeeLine

### Загальна інстркуція

вам треба фронт і бек у відповідних директоріях. 
- бек треба або покласти в директорію ```api```, або замінти цю директорію на свою, тобто всередині диреткорії апі вже має лежати соурскод, не треба робити умвоно таку стркутуру ```api/CoffeeLineApi/{some root files}```, а просто ```api/{some root files}```.
- фронт аналогічно, просто всі файли закинути в ```frontend```
